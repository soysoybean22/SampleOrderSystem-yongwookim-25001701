---

## 3. 컬럼 너비 계산 기준

모든 테이블: 전체 120자 (내부 118자).  
한글 헤더에는 `padRight()` 적용 (Phase 7에서 확립한 방식).  
ANSI 코드 포함 데이터 행(상태 배지, Progress Bar)은 표시 너비를 별도 계산해 trailing 공백으로 맞춘다.

## 4. 화면별 상세 변경

### 4-1. showStockStatus() — 재고량 확인

#### 현재 구조

```
  시료명                                    재고    미처리주문   상태        잔여율
  ────────────────────────────────────────────────────────────────────────────────
  실리콘 웨이퍼-8인치             480 ea    100 ea  여유        [████████░░]  80%
```

#### 변경 후 구조

```
┌──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│  시료명                                    재고    미처리주문   상태        잔여율                                    │
├──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│  실리콘 웨이퍼-8인치             480 ea    100 ea  여유        [████████░░]  80%                                      │
│  GaN 에피택셜-4인치              220 ea    500 ea  부족        [████░░░░░░]  44%                                      │
└──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```

#### 컬럼 배치 (내부 118)

| 컬럼 | 표시 너비 | 비고 |
|------|---------|------|
| 시료명 | 38 display | `padRight(name, 38)` |
| 재고 | 9 display | `%6d ea` |
| 미처리주문 | 11 display | `%8d ea` |
| 상태 | 4 display | ANSI — 항상 2자 한글 = 4 display |
| Progress Bar | 12 display | ANSI — 항상 `[` + 10칸 + `]` = 12 display |
| 잔여율 | 4 display | `%3d%%` |
| trailing | 30 | 118 − (2+38+1+9+1+11+2+4+2+12+2+4) |

**헤더 행:**
```java
ConsoleHelper.println("│  " + ConsoleHelper.padRight("시료명", 38)
    + " " + ConsoleHelper.padRight("재고", 9)
    + " " + ConsoleHelper.padRight("미처리주문", 11)
    + "  " + ConsoleHelper.padRight("상태", 4)
    + "  " + ConsoleHelper.padRight("잔여율", 18)
    + " ".repeat(30) + "│");
```

> `잔여율` 헤더는 Progress Bar(12) + `  ` (2) + ratio(4) = 18 display 영역을 포함해 `padRight("잔여율", 18)`.  
> 검증: 3+38+1+9+1+11+2+4+2+18+30+1 = **120** ✓

**데이터 행:**
```java
String dataLine = String.format("  %s %6d ea %8d ea  %-10s  %s  %3d%%",
    ConsoleHelper.padRight(name, 38), stock, pending,
    AnsiColor.stockStatusColored(status), ConsoleHelper.progressBar(ratio), ratio);
ConsoleHelper.println("│" + dataLine + " ".repeat(30) + "│");
```

> `%-10s` 자리의 `stockStatusColored`는 ANSI 포함 Java 문자열이 10자 이상이므로 별도 패딩 없음.  
> 실제 표시 너비는 항상 4 (2자 한글). 30자 trailing이 나머지를 채운다.  
> 검증: 1 + 2+38+1+9+1+11+2+4+2+12+2+4 + 30 + 1 = **120** ✓

**풋노트** (`* 미처리 주문 = ...`): 테이블 밖, 기존 WARN 색상 유지.

### 4-2. showOrderSummary() — 주문량 확인

#### 현재 구조

```
  상태별 주문 현황

  [RESERVED]      3건
  ...
  ─────────────────────
  합계            10건  (REJECTED 제외)
```

#### 변경 후 구조

```
┌──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│  상태별 주문 현황                                                                                                    │
├──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│  [RESERVED]      3건                                                                                                 │
│  [PRODUCING]     2건  ← 생산라인 대기                                                                                │
│  ...                                                                                                                 │
├──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│  합계            10건  (REJECTED 제외)                                                                                │
└──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```

#### trailing 계산 방식

status badge 표시 너비 = `[XXXXX]` 텍스트 길이 = `entry.getKey().name().length() + 2`.

```java
int badgeLen = entry.getKey().name().length() + 2;  // ex) RESERVED→10, CONFIRMED→11
int suffixLen = entry.getKey() == OrderStatus.PRODUCING ? 16 : 0;
int trailing = 118 - (2 + badgeLen + 1 + 6 + suffixLen);
ConsoleHelper.println("│  " + badge + String.format(" %4d건", count) + suffix
    + " ".repeat(trailing) + "│");
```

**헤더 행 (테이블 제목):**
```java
ConsoleHelper.println("│  " + ConsoleHelper.padRight("상태별 주문 현황", 116) + "│");
```

**합계 행:**  
`"합계"` 표시 너비 = 4 display.  
`" %4d건  (REJECTED 제외)"` = 1+4+2+2+1+8+1+4 = 23 display.  
trailing = 118 − (2+4+1+23) = 118 − 30 = 88.

```java
ConsoleHelper.println("│  " + ConsoleHelper.padRight("합계", 4)
    + String.format(" %4d건  (REJECTED 제외)", total)
    + " ".repeat(88) + "│");
```

> 검증: 3+4+23+88+1 = **119**... padRight("합계", 5)로 1 추가:
> 3+5+23+88+1 = **120** ✓

**풋노트** (`* 거절된 주문 ...`): 테이블 밖 유지.

### 4-3. printWaitingQueue() — 생산라인 대기 큐

#### 현재 구조

```
[ 대기 중인 주문 (FIFO 순) ]
  순서  주문번호               시료명                                    부족분  실생산량    총생산시간
  ────────────────────────────────────────────────────────────────────────────────
  1     ORD-20260416-0008    산화막 웨이퍼-SiO2                        150 ea  190 ea    114.0 min
```

#### 변경 후 구조

```
[ 대기 중인 주문 (FIFO 순) ]
┌──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│  순서  주문번호                시료명                                   부족분    실생산량   총생산시간                  │
├──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│  1     ORD-20260416-0008    산화막 웨이퍼-SiO2                         150 ea   190 ea   114.0 min                    │
└──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```

#### 컬럼 배치 (내부 118)

| 컬럼 | 표시 너비 | 비고 |
|------|---------|------|
| 순서 | 4 | `%-4d` / `padRight("순서", 4)` |
| 주문번호 | 22 | `%-22s` (ASCII only) |
| 시료명 | 40 | `padRight(name, 40)` |
| 부족분 | 8 | `%5d ea` |
| 실생산량 | 9 | `%6d ea` |
| 총생산시간 | 11 | `%7.1f min` |
| trailing | 17 | 118 − (2+4+1+22+1+40+1+8+1+9+1+11) |

검증: 2+4+1+22+1+40+1+8+1+9+1+11+17 = 118 → `│  content  │` = **120** ✓

**헤더 행:**
```java
ConsoleHelper.println("│  " + ConsoleHelper.padRight("순서", 4)
    + " " + ConsoleHelper.padRight("주문번호", 22)
    + " " + ConsoleHelper.padRight("시료명", 40)
    + " " + ConsoleHelper.padRight("부족분", 8)
    + " " + ConsoleHelper.padRight("실생산량", 9)
    + " " + ConsoleHelper.padRight("총생산시간", 11)
    + " ".repeat(17) + "│");
```

**데이터 행:**
```java
System.out.printf("│  %-4d %-22s %s %5d ea %6d ea %7.1f min" + " ".repeat(17) + "│%n",
    i, j.getOrderId(),
    ConsoleHelper.padRight(resolveSampleName(j.getSampleId()), 40),
    j.getShortage(), j.getActualProductionQty(), j.getTotalProductionTime());
```
